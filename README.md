## Project Title
Student Portal System

## Project Description
The Student Portal System is a simple web-based application developed using Next.js
to demonstrate Software Configuration Management (SCM) practices. The system allows
students to log in, access a dashboard, and view basic academic information.

The focus of this project is on applying SCM processes such as version control,
change management, baselining, and configuration audits rather than full system
functionality.

## System Features
- Student login page
- Main dashboard page displaying services
- View registered courses (core functionality)
- Simple data storage using a JSON file

## Technologies Used
- Next.js (React framework)
- JavaScript
- HTML / CSS
- Git & GitHub
